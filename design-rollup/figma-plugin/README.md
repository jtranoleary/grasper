# Figma Plugin (Vanilla JS)

## Getting Started

1.  Open Figma.
2.  Go to **Plugins** > **Development** > **Import plugin from manifest...**.
3.  Select the `manifest.json` file in this directory.

## File Structure

-   `manifest.json`: The plugin manifest file.
-   `code.js`: The main logic for the plugin.
-   `ui.html`: The user interface for the plugin (if applicable).
